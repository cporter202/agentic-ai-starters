# Curated API Stack

## Featured APIs

| API | Used for | Why it belongs |
| --- | --- | --- |
| [LinkedIn Jobs Scraper](https://apify.com/valig/linkedin-jobs-scraper?fpr=p2hrc6) | LinkedIn job discovery | Strong fit for high-signal white-collar searches |
| [Indeed Jobs Scraper](https://apify.com/valig/indeed-jobs-scraper?fpr=p2hrc6) | broad job-market collection | Useful for wider role coverage and geographic depth |
| [LinkedIn Company Data Extractor](https://apify.com/data-slayer/linkedin-company-scraper?fpr=p2hrc6) | employer enrichment | Helps candidates evaluate the company behind the listing |
| [Website Content Crawler](https://apify.com/apify/website-content-crawler?fpr=p2hrc6) | careers-page and employer-site extraction | Adds richer company context for prioritization and prep |

## Companion APIs

- OpenAI: fit scoring, tailored application assets, and reminders.
- Notion API: pipeline tracking.
- Resend: follow-up drafts or alerts.

## Suggested stack tiers

- Lean stack: OpenAI + [Indeed Jobs Scraper](https://apify.com/valig/indeed-jobs-scraper?fpr=p2hrc6).
- Best-value stack: OpenAI + [LinkedIn Jobs Scraper](https://apify.com/valig/linkedin-jobs-scraper?fpr=p2hrc6) + [LinkedIn Company Data Extractor](https://apify.com/data-slayer/linkedin-company-scraper?fpr=p2hrc6).
- Premium stack: OpenAI + [LinkedIn Jobs Scraper](https://apify.com/valig/linkedin-jobs-scraper?fpr=p2hrc6) + [Indeed Jobs Scraper](https://apify.com/valig/indeed-jobs-scraper?fpr=p2hrc6) + Notion API + Langfuse.
