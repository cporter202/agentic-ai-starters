# Architecture

## Core components

- candidate profile intake
- job source collector
- job normalization and deduplication
- fit-scoring agent
- application-material generator
- pipeline tracker and reminder engine

## Suggested flow

1. Convert the candidate profile into structured fit criteria.
2. Collect jobs from one or more sources and normalize the schema.
3. Score each role for alignment, compensation, and likely value.
4. Generate tailored materials for top matches.
5. Track status changes, deadlines, and follow-up tasks.

## MVP notes

- Start with one niche or role family so the scoring model stays sharp.
- Keep generated assets editable so candidates stay in control of the final voice.
- Save rejection reasons and interview outcomes to improve future ranking.
