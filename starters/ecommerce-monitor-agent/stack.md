# Curated API Stack

## Featured APIs

| API | Used for | Why it belongs |
| --- | --- | --- |
| [Amazon Product Scraper](https://apify.com/get-leads/amazon-product-scraper?fpr=p2hrc6) | marketplace catalog, price, and review monitoring | Strong fit for Amazon-focused operators and comparison tracking |
| [Google Search API](https://apify.com/api/google-search-api?fpr=p2hrc6) | shopping and brand SERP checks | Useful for watching visibility and competitor movement outside owned channels |
| [Website Content Crawler](https://apify.com/apify/website-content-crawler?fpr=p2hrc6) | competitor store-page crawling | Helps monitor product-page, pricing, and assortment changes |
| [Trustpilot Scraper](https://apify.com/happitap/trustpilot-scraper?fpr=p2hrc6) | review harvesting | Useful for voice-of-customer and reputation tracking across brands |

## Companion APIs

- OpenAI: summarization, anomaly triage, and operator recommendations.
- Shopify Admin API: owned-store catalog sync.
- Slack or Resend: alert delivery.

## Suggested stack tiers

- Lean stack: OpenAI + [Amazon Product Scraper](https://apify.com/get-leads/amazon-product-scraper?fpr=p2hrc6).
- Best-value stack: OpenAI + [Amazon Product Scraper](https://apify.com/get-leads/amazon-product-scraper?fpr=p2hrc6) + [Website Content Crawler](https://apify.com/apify/website-content-crawler?fpr=p2hrc6).
- Premium stack: OpenAI + Shopify Admin API + [Amazon Product Scraper](https://apify.com/get-leads/amazon-product-scraper?fpr=p2hrc6) + [Trustpilot Scraper](https://apify.com/happitap/trustpilot-scraper?fpr=p2hrc6) + Langfuse.
