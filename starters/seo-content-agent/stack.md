# Curated API Stack

## Featured Apify APIs

| Apify API | Used for | Why it belongs |
| --- | --- | --- |
| [Google Search API](https://apify.com/api/google-search-api?fpr=p2hrc6) | SERP collection and intent research | Gives the agent current search-result context without building the fetch layer yourself |
| [Google Search Scraper](https://apify.com/apidojo/google-search-scraper?fpr=p2hrc6) | deeper SERP extraction | Useful when you want page-level search data for briefs and content gaps |
| [Google Keyword Scraper](https://apify.com/dxbear/google-keyword-scraper?fpr=p2hrc6) | keyword ideation and SEO metrics | Strong fit for turning a seed topic into clusters and article opportunities |
| [Website Content Crawler](https://apify.com/apify/website-content-crawler?fpr=p2hrc6) | competitor-page extraction | Helps compare outlines, FAQs, and page depth before drafting |

## Companion APIs

- OpenAI: outlining, drafting, editing, and structured QA.
- Notion API: editorial handoff and approval.
- Langfuse: prompt evaluation and content workflow traces.

## Suggested stack tiers

- Lean stack: OpenAI + [Google Keyword Scraper](https://apify.com/dxbear/google-keyword-scraper?fpr=p2hrc6).
- Best-value stack: OpenAI + [Google Search API](https://apify.com/api/google-search-api?fpr=p2hrc6) + [Website Content Crawler](https://apify.com/apify/website-content-crawler?fpr=p2hrc6).
- Premium stack: OpenAI + [Google Search Scraper](https://apify.com/apidojo/google-search-scraper?fpr=p2hrc6) + [Google Keyword Scraper](https://apify.com/dxbear/google-keyword-scraper?fpr=p2hrc6) + Langfuse.
