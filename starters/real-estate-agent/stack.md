# Curated API Stack

## Featured APIs

| API | Used for | Why it belongs |
| --- | --- | --- |
| [Zillow Scraper](https://apify.com/mido_99/zillow-scraper?fpr=p2hrc6) | property listing collection | Gives the starter a direct path to listing intelligence |
| [Google Maps B2B Leads Scraper](https://apify.com/primeparse/google-maps-scraper?fpr=p2hrc6) | nearby-business and amenity context | Useful for neighborhood enrichment and local operator research |
| [Google Search API](https://apify.com/api/google-search-api?fpr=p2hrc6) | market and locality discovery | Helps monitor local signals, neighborhoods, and market chatter |
| [Website Content Crawler](https://apify.com/apify/website-content-crawler?fpr=p2hrc6) | brokerage or local-site extraction | Useful when the agent needs more local detail than listing portals provide |

## Companion APIs

- OpenAI: screening notes, underwriting summaries, and operator packets.
- RentCast API: rent estimates and structured property data.
- Resend: deal alerts and daily digests.

## Suggested stack tiers

- Lean stack: OpenAI + [Zillow Scraper](https://apify.com/mido_99/zillow-scraper?fpr=p2hrc6).
- Best-value stack: OpenAI + [Zillow Scraper](https://apify.com/mido_99/zillow-scraper?fpr=p2hrc6) + RentCast API.
- Premium stack: OpenAI + [Zillow Scraper](https://apify.com/mido_99/zillow-scraper?fpr=p2hrc6) + [Google Maps B2B Leads Scraper](https://apify.com/primeparse/google-maps-scraper?fpr=p2hrc6) + Langfuse.
